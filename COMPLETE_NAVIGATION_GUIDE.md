## 🗺️ خريطة التنقل السريع

### للوصول للاختبارات:
```
1. افتح http://localhost:3000/dashboard
2. من القائمة الجانبية: إدارة المنصة
3. اضغط على "الاختبارات"
```

### لإضافة أسئلة لاختبار:
```
1. افتح http://localhost:3000/dashboard/admin/quizzes
2. اضغط على أيقونة العين (👁️) بجانب أي اختبار
3. اضغط "إضافة سؤال"
```

### لإدارة الدروس:
```
1. افتح http://localhost:3000/dashboard/courses
2. اختر دورة واضغط "تعديل"
3. ستجد قسم "الدروس" في الأعلى
4. اضغط "إضافة درس"
```

### للتسجيل اليدوي:
```
1. افتح http://localhost:3000/dashboard/admin/students
2. اختر طالب
3. اضغط أيقونة "تسجيل في دورة" (UserPlus)
4. املأ البيانات واحفظ
```
