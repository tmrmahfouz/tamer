## 🎯 الأنظمة الرئيسية

### 1. نظام التسجيل اليدوي للطلاب 👥
**الصفحة:** `/dashboard/admin/students`

**كيفية الوصول:**
- من القائمة الجانبية: **إدارة المنصة** → **إدارة الطلاب**
- من الإجراءات السريعة: **تسجيل طالب يدوياً**

**الميزات:**
- ✅ إضافة طالب جديد
- ✅ تسجيل طالب في دورة يدوياً
- ✅ اختيار طريقة الدفع
- ✅ تحديد المبلغ المدفوع
- ✅ إضافة ملاحظات

**خطوات التسجيل اليدوي:**
```
1. اذهب إلى /dashboard/admin/students
2. اختر الطالب المطلوب
3. اضغط على أيقونة "تسجيل في دورة" (UserPlus)
4. اختر الدورة
5. حدد المبلغ المدفوع
6. اختر طريقة الدفع (يدوي، نقدي، فودافون كاش، إلخ)
7. أضف ملاحظات (اختياري)
8. اضغط "تسجيل في الدورة"
```

---

### 2. نظام قبول المدفوعات 💰
**الصفحة:** `/dashboard/admin/payments`

**كيفية الوصول:**
- من القائمة الجانبية: **إدارة المنصة** → **المدفوعات**
- من الإجراءات السريعة: **مراجعة المدفوعات**

**الميزات:**
- ✅ عرض جميع المدفوعات المعلقة
- ✅ مراجعة إيصالات الدفع
- ✅ تأكيد الدفع (Verify)
- ✅ رفض الدفع (Reject)
- ✅ فلترة حسب الحالة
- ✅ تصدير المدفوعات

**خطوات قبول الدفع:**
```
1. اذهب إلى /dashboard/admin/payments
2. اختر "معلق" من الفلتر
3. شاهد تفاصيل الدفع
4. راجع إيصال الدفع
5. اضغط "تأكيد" لقبول الدفع
6. يتم تفعيل التسجيل تلقائياً
```

---

### 3. نظام بوابات الدفع 💳
**الصفحة:** `/dashboard/admin/payment-gateways`

**كيفية الوصول:**
- من القائمة الجانبية: **إدارة المنصة** → **بوابات الدفع**
- من الإجراءات السريعة: **بوابات الدفع**

**البوابات المتاحة:**

#### 1. فودافون كاش 📱
```
رقم الحساب: 01012345678
اسم الحساب: تامر محفوظ
الرسوم: 0%
```

#### 2. انستاباي 💳
```
اسم المستخدم: @tamermahfouz
الرسوم: 0%
```

#### 3. فوري 🏪
```
كود التاجر: 12345
الرسوم: 5 جنيه
```

#### 4. تحويل بنكي 🏦
```
البنك: البنك الأهلي المصري
اسم الحساب: تامر محفوظ
رقم الحساب: 1234567890
IBAN: EG380002000156789012345180002
الرسوم: 0%
```

**الميزات:**
- ✅ عرض جميع بوابات الدفع
- ✅ تفاصيل كل بوابة
- ✅ معلومات الحساب
- ✅ التعليمات للطلاب
- ✅ الرسوم

---

### 4. نظام الكوبونات 🎟️
**الصفحة:** `/dashboard/admin/coupons`

**كيفية الوصول:**
- من القائمة الجانبية: **إدارة المنصة** → **الكوبونات**
- من الإجراءات السريعة: **إدارة الكوبونات**

**الميزات:**
- ✅ إنشاء كوبون جديد
- ✅ خصم نسبة مئوية أو مبلغ ثابت
- ✅ تحديد دورات معينة
- ✅ حد أقصى للاستخدام
- ✅ تاريخ انتهاء
- ✅ تفعيل/إيقاف
- ✅ إحصائيات الاستخدام

**خطوات إنشاء كوبون:**
```
1. اذهب إلى /dashboard/admin/coupons
2. اضغط "إضافة كوبون"
3. أدخل كود الكوبون (مثل: SUMMER2024)
4. اختر نوع الخصم (نسبة أو مبلغ ثابت)
5. حدد قيمة الخصم
6. حدد الحد الأقصى للاستخدام (0 = غير محدود)
7. اختر تاريخ الانتهاء
8. فعّل الكوبون
9. احفظ
```

## 🚀 سير العمل الموصى به

### للAdmin اليومي:
```
1. افتح /dashboard
2. شاهد الإحصائيات السريعة

4. اذهب إلى /dashboard/admin/payments
5. راجع وقبول المدفوعات المعلقة

7. راجع وقبول التسجيلات المعلقة

5. اضغط "تسجيل في دورة"
6. اختر الدورة والمبلغ
7. سجّل
```

```
1. /dashboard/admin/coupons
2. اضغط "إضافة كوبون"

4. احفظ
```
