# 📝 دليل نظام الاختبارات

## 🎯 نظرة عامة

نظام اختبارات متكامل يتيح للمدرسين والأدمن إنشاء وإدارة اختبارات للدورات والدروس مع إمكانيات متقدمة.

---

## 📍 الوصول لقسم الاختبارات

### من لوحة التحكم:
```
/dashboard → إدارة المنصة → الاختبارات
```

### مباشرة:
```
/dashboard/admin/quizzes
```

- متوسط عدد الأسئلة

## 📝 خطوات إنشاء اختبار

### 1. افتح صفحة الاختبارات:
```
/dashboard/admin/quizzes
```

### 2. اضغط "إضافة اختبار":
- الزر في أعلى يمين الصفحة

### 3. املأ البيانات الأساسية:

#### الحقول الإلزامية:
- **عنوان الاختبار** ⭐ (مطلوب)
- **الدورة** ⭐ (مطلوب)

#### الحقول الاختيارية:
- **الوصف**: وصف مختصر للاختبار
- **الدرس**: اختر درس محدد أو اتركه فارغاً للاختبار العام
- **درجة النجاح**: النسبة المئوية المطلوبة للنجاح (افتراضي: 70%)
- **الوقت**: الوقت المحدد بالدقائق (افتراضي: 30 دقيقة)
- **عدد المحاولات**: عدد المحاولات المسموحة (افتراضي: 3)
- **نشر الاختبار**: ✓ لجعل الاختبار متاح للطلاب

### 4. احفظ الاختبار:
- اضغط "إنشاء"
- سيتم إنشاء الاختبار بدون أسئلة
- يمكنك إضافة الأسئلة لاحقاً

---

## ✏️ إضافة الأسئلة

### بعد إنشاء الاختبار:
```
1. اضغط على أيقونة العين (👁️) بجانب الاختبار
2. ستفتح صفحة تفاصيل الاختبار
3. اضغط "إضافة سؤال"
4. املأ بيانات السؤال:
   - نص السؤال
   - نوع السؤال (اختيار متعدد، صح/خطأ، إجابة قصيرة)
   - الخيارات (للاختيار المتعدد)
   - الإجابة الصحيحة
   - النقاط
   - التوضيح (اختياري)
5. احفظ السؤال
```

---

## 🔄 تعديل اختبار

### 1. في قائمة الاختبارات:
- اضغط على أيقونة القلم (✏️) بجانب الاختبار

### 2. عدّل البيانات:
- جميع الحقول قابلة للتعديل
- يمكنك تغيير الدورة أو الدرس
- يمكنك تعديل الإعدادات

### 3. احفظ التعديلات:
- اضغط "تحديث"
- سيتم تحديث الاختبار فوراً

---

## 🗑️ حذف اختبار

### 1. في قائمة الاختبارات:
- اضغط على أيقونة سلة المهملات (🗑️) بجانب الاختبار

### 2. تأكيد الحذف:
- سيظهر تأكيد: "هل أنت متأكد من حذف هذا الاختبار؟"
- اضغط "موافق" للحذف

### 3. النتيجة:
- سيتم حذف الاختبار نهائياً
- سيتم حذف جميع الأسئلة المرتبطة
- سيتم حذف جميع محاولات الطلاب

---

## 📊 أنواع الاختبارات

### 1. اختبار عام للدورة:
```
- لا يرتبط بدرس محدد
- يظهر في صفحة الدورة الرئيسية
- يمكن أن يكون اختبار نهائي
```

### 2. اختبار خاص بدرس:
```
- يرتبط بدرس محدد
- يظهر في نهاية الدرس
- يجب إكمال الدرس أولاً
```

## 🔄 سير العمل الموصى به

### لإنشاء اختبار كامل:

#### المرحلة 1: إنشاء الاختبار
```
1. اذهب إلى /dashboard/admin/quizzes
2. اضغط "إضافة اختبار"
3. املأ البيانات الأساسية
4. احفظ كمسودة (غير منشور)
```

#### المرحلة 2: إضافة الأسئلة
```
1. افتح تفاصيل الاختبار
2. أضف الأسئلة واحداً تلو الآخر
3. راجع الأسئلة والإجابات
4. تأكد من النقاط والتوضيحات
```

#### المرحلة 3: المراجعة
```
1. راجع جميع الأسئلة
2. تأكد من الإعدادات
3. اختبر الاختبار (إن أمكن)
```

#### المرحلة 4: النشر
```
1. عدّل الاختبار
2. فعّل خيار "نشر الاختبار"
3. احفظ التغييرات
4. الاختبار أصبح متاح للطلاب
```

## 🛠️ الـ APIs المستخدمة

### 1. جلب جميع الاختبارات:
```typescript
GET /api/quizzes

Response:
{
  success: true,
  count: 25,
  quizzes: [...]
}
```

### 2. إنشاء اختبار جديد:
```typescript
POST /api/quizzes

Body:
{
  title: "عنوان الاختبار",
  description: "وصف الاختبار",
  course: "course_id",
  lesson: "lesson_id", // اختياري
  passingScore: 70,
  timeLimit: 30,
  maxAttempts: 3,
  isPublished: false
}

Response:
{
  success: true,
  message: "تم إنشاء الاختبار بنجاح",
  quiz: {...}
}
```

### 3. تعديل اختبار:
```typescript
PUT /api/quizzes/[id]

Body:
{
  title: "عنوان محدث",
  ...
}

Response:
{
  success: true,
  message: "تم تحديث الاختبار بنجاح",
  quiz: {...}
}
```

### 4. حذف اختبار:
```typescript
DELETE /api/quizzes/[id]

Response:
{
  success: true,
  message: "تم حذف الاختبار بنجاح"
}
```

## 💡 نصائح مهمة

### 1. تنظيم الاختبارات:
- استخدم أسماء واضحة ومميزة
- اربط كل اختبار بالدرس المناسب
- نظم الاختبارات حسب الصعوبة

### 2. إعدادات الوقت:
- اختبارات قصيرة: 10-15 دقيقة
- اختبارات متوسطة: 20-30 دقيقة
- اختبارات نهائية: 45-60 دقيقة

### 3. درجة النجاح:
- اختبارات تقييمية: 50-60%
- اختبارات عادية: 70-75%
- اختبارات نهائية: 80-85%

### 4. عدد المحاولات:
- اختبارات تدريبية: غير محدود
- اختبارات عادية: 2-3 محاولات
- اختبارات نهائية: محاولة واحدة
