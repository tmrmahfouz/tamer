## 🚀 كيف تبدأ؟

### الخطوة 1: إعداد MongoDB

**اختر أحد الخيارين:**

#### الخيار أ: MongoDB Atlas (سهل وسريع - موصى به)
1. اذهب إلى: https://www.mongodb.com/cloud/atlas/register
2. سجل حساب مجاني
3. أنشئ Cluster (اختر FREE)
4. احصل على Connection String
5. عدّل `.env.local`:
   ```env
   MONGODB_URI=your-connection-string-here
   ```

📖 **دليل تفصيلي:** [MONGODB_SETUP.md](MONGODB_SETUP.md)

#### الخيار ب: MongoDB محلي
1. حمّل من: https://www.mongodb.com/try/download/community
2. ثبّت البرنامج
3. شغّل: `net start MongoDB`
4. `.env.local` جاهز بالفعل!

---

### الخطوة 2: إنشاء البيانات التجريبية

```bash
npm run seed
```

ستحصل على:
- 👤 5 مستخدمين (مدير + مدرس + 3 طلاب)
- 📚 6 دورات تدريبية كاملة

**بيانات تسجيل الدخول:**
```
المدير:
  📧 admin@tamermahfouz.com
  🔑 123456

المدرس:
  📧 instructor@tamermahfouz.com
  🔑 123456

الطالب:
  📧 student1@example.com
  🔑 123456
```

---

### الخطوة 3: تشغيل المشروع

```bash
npm run dev
```

افتح المتصفح: **http://localhost:3000**
