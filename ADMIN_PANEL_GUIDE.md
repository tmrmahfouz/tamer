## 🔌 APIs المتاحة

### إدارة المعلمين:
```
GET    /api/admin/instructors          - جلب جميع المعلمين
POST   /api/admin/instructors          - إضافة معلم
PUT    /api/admin/instructors/[id]     - تعديل معلم
DELETE /api/admin/instructors/[id]     - حذف معلم
```

### إدارة الطلاب:
```
GET    /api/users?role=student          - جلب جميع الطلاب
POST   /api/admin/students              - إضافة طالب
PUT    /api/admin/students/[id]         - تعديل طالب
DELETE /api/admin/students/[id]         - حذف طالب
```

### إدارة التسجيلات:
```
GET    /api/admin/enrollments                  - جلب جميع التسجيلات
POST   /api/admin/enrollments/[id]/approve     - الموافقة على تسجيل
POST   /api/admin/enrollments/[id]/reject      - رفض تسجيل
```

### إدارة المدفوعات:
```
GET    /api/admin/payments                - جلب جميع المدفوعات
POST   /api/admin/payments/[id]/verify    - تأكيد دفعة
POST   /api/admin/payments/[id]/reject    - رفض دفعة
```

### التحليلات:
```
GET    /api/admin/analytics?period=month  - جلب الإحصائيات
```
