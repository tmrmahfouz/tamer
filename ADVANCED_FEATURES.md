## 🎯 الأنظمة المضافة

### 1. نظام الكوبونات والخصومات 🎟️
**الصفحة:** `/dashboard/admin/coupons`

**الميزات:**
- ✅ إنشاء كوبونات خصم نسبة مئوية أو مبلغ ثابت
- ✅ تحديد دورات معينة للكوبون
- ✅ حد أقصى لعدد مرات الاستخدام
- ✅ تاريخ انتهاء الصلاحية
- ✅ تفعيل/إيقاف الكوبونات
- ✅ إحصائيات شاملة (إجمالي، نشطة، منتهية، مرات الاستخدام)
- ✅ بحث وفلترة متقدمة

**APIs:**
```
GET    /api/admin/coupons          - جلب جميع الكوبونات
POST   /api/admin/coupons          - إنشاء كوبون جديد
PUT    /api/admin/coupons/[id]     - تحديث كوبون
DELETE /api/admin/coupons/[id]     - حذف كوبون
POST   /api/coupons/validate       - التحقق من صلاحية كوبون
```

## 🚀 كيفية الاستخدام

### 1. تسجيل الدخول كـ Admin:
```
البريد: admin@tamermahfouz.com
كلمة المرور: 123456
```

### 2. استكشاف الأنظمة الجديدة:
- افتح القائمة الجانبية
- ستجد قسم "إدارة المنصة" مع جميع الأدوات
- كل صفحة تحتوي على واجهة سهلة الاستخدام

### 3. إنشاء كوبون:
1. اذهب إلى `/dashboard/admin/coupons`
2. اضغط "إضافة كوبون"
3. املأ البيانات
4. احفظ

### 4. عرض الإحصائيات:
1. اذهب إلى `/dashboard/admin/statistics`
2. اختر الفترة الزمنية
3. استعرض الرسوم البيانية والتحليلات

### 5. إنشاء نسخة احتياطية:
1. اذهب إلى `/dashboard/admin/backup`
2. اختر المجموعات
3. اضغط "إنشاء نسخة احتياطية"
4. احفظ الملف في مكان آمن
