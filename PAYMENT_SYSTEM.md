# 💳 نظام الدفع والمحافظ الإلكترونية

## 📊 نظرة عامة

تم إضافة نظام دفع شامل يدعم جميع طرق الدفع المصرية الشهيرة مع إمكانية إدارة الطلاب وتسجيلهم يدوياً في الدورات.

---

## 🎯 الأنظمة المضافة

### 1. نظام إدارة الطلاب المتقدم 👥

**الصفحة:** `/dashboard/admin/students`

**الميزات:**
- ✅ عرض جميع الطلاب مع إحصائيات شاملة
- ✅ إضافة طالب جديد يدوياً
- ✅ تسجيل الطالب في دورة يدوياً
- ✅ عرض عدد الدورات المسجل فيها كل طالب
- ✅ بحث متقدم بالاسم والبريد الإلكتروني
- ✅ حذف الطلاب
- ✅ إحصائيات (إجمالي، نشطون، جدد هذا الشهر)

**APIs:**
```
GET    /api/admin/students                 - جلب جميع الطلاب مع الإحصائيات
POST   /api/admin/students                 - إضافة طالب جديد
PUT    /api/admin/students/[id]            - تحديث بيانات طالب
DELETE /api/admin/students/[id]            - حذف طالب
POST   /api/admin/enrollments/manual       - تسجيل طالب يدوياً في دورة
```

---

### 2. نظام التسجيل اليدوي في الدورات 📝

**الميزات:**
- ✅ تسجيل أي طالب في أي دورة من لوحة التحكم
- ✅ تحديد المبلغ المدفوع
- ✅ اختيار طريقة الدفع
- ✅ إضافة ملاحظات
- ✅ إنشاء سجل دفع تلقائي
- ✅ تحديث عدد الطلاب في الدورة

**طرق الدفع المتاحة:**
- `manual` - يدوي (Admin)
- `cash` - نقدي
- `bank_transfer` - تحويل بنكي
- `vodafone_cash` - فودافون كاش
- `instapay` - انستاباي
- `fawry` - فوري

**API:**
```
POST /api/admin/enrollments/manual

Body:
{
  "studentId": "student_id",
  "courseId": "course_id",
  "paymentAmount": 500,
  "paymentMethod": "vodafone_cash",
  "notes": "تسجيل يدوي"
}
```

---

### 3. نظام بوابات الدفع المصرية 💰

**الصفحة:** `/dashboard/admin/payment-gateways`

**البوابات المدعومة:**

#### 1. فودافون كاش 📱
- **النوع:** محفظة إلكترونية
- **المعلومات المطلوبة:**
  - رقم الحساب
  - اسم الحساب
- **التعليمات:** تحويل المبلغ + إرسال صورة الإيصال

#### 2. انستاباي 💳
- **النوع:** تحويل فوري
- **المعلومات المطلوبة:**
  - اسم المستخدم
- **التعليمات:** التحويل + إرسال رقم المرجع

#### 3. فوري 🏪
- **النوع:** نقاط بيع
- **المعلومات المطلوبة:**
  - كود التاجر
- **التعليمات:** الدفع في أقرب فرع + إرسال رقم المرجع

#### 4. تحويل بنكي 🏦
- **النوع:** تحويل بنكي
- **المعلومات المطلوبة:**
  - اسم البنك
  - اسم الحساب
  - رقم الحساب
  - IBAN
- **التعليمات:** التحويل البنكي + إرسال صورة الإيصال

**Model:**
```typescript
interface PaymentGateway {
  name: string
  type: 'vodafone_cash' | 'instapay' | 'fawry' | 'bank_transfer' | 'wallet'
  isActive: boolean
  config: {
    accountNumber?: string
    accountName?: string
    bankName?: string
    iban?: string
    merchantCode?: string
    instructions?: string
  }
  fees: {
    type: 'percentage' | 'fixed'
    value: number
  }
  minAmount?: number
  maxAmount?: number
}
```

### 4. صفحة الدفع للطلاب 🛒

**الصفحة:** `/courses/[id]/checkout`

**الميزات:**
- ✅ عرض ملخص الدورة والسعر
- ✅ اختيار طريقة الدفع
- ✅ عرض تفاصيل البوابة المختارة
- ✅ إدخال رقم الهاتف
- ✅ إدخال رقم المرجع
- ✅ رفع صورة إيصال الدفع
- ✅ إرسال طلب الدفع للمراجعة
- ✅ واجهة مستخدم احترافية

**سير العمل:**
1. الطالب يختار الدورة
2. يضغط على "اشترك الآن"
3. يتم توجيهه لصفحة الدفع
4. يختار طريقة الدفع
5. يرى تفاصيل الحساب/الرقم
6. يقوم بالدفع خارج المنصة
7. يرفع إثبات الدفع
8. يرسل الطلب
9. Admin يراجع ويوافق
10. يتم تفعيل التسجيل

---

### 5. نظام قبول المدفوعات 📋

**الصفحة:** `/dashboard/admin/payments`

**الميزات:**
- ✅ عرض جميع طلبات الدفع
- ✅ فلترة حسب الحالة (معلق، مكتمل، مرفوض)
- ✅ عرض تفاصيل الدفع
- ✅ عرض صورة الإيصال
- ✅ تأكيد الدفع
- ✅ رفض الدفع
- ✅ إحصائيات شاملة

**APIs:**
```
GET  /api/admin/payments                 - جلب جميع المدفوعات
POST /api/admin/payments/[id]/verify     - تأكيد دفعة
POST /api/admin/payments/[id]/reject     - رفض دفعة
```

---

## 🔄 سير العمل الكامل

### للطالب:
1. **التصفح:** يتصفح الدورات المتاحة
2. **الاختيار:** يختار دورة ويضغط "اشترك الآن"
3. **الدفع:** يختار طريقة الدفع ويدفع
4. **الإثبات:** يرفع صورة الإيصال
5. **الانتظار:** ينتظر موافقة Admin
6. **البدء:** يبدأ الدراسة بعد الموافقة

### للAdmin:
1. **المراجعة:** يراجع طلبات الدفع المعلقة
2. **التحقق:** يتحقق من صورة الإيصال
3. **القرار:** يوافق أو يرفض
4. **التفعيل:** يتم تفعيل التسجيل تلقائياً عند الموافقة

### التسجيل اليدوي:
1. **الاختيار:** Admin يختار الطالب
2. **التسجيل:** يضغط "تسجيل في دورة"
3. **التفاصيل:** يختار الدورة والمبلغ وطريقة الدفع
4. **التأكيد:** يتم التسجيل فوراً
