# 📖 دليل محرر الصفحة الرئيسية

## 🎯 نظرة عامة

محرر الصفحة الرئيسية هو أداة قوية تتيح لك:
- ✅ استيراد جميع الأقسام الحالية من الصفحة الرئيسية
- ✅ إضافة أقسام جديدة
- ✅ تحرير كل قسم بشكل منفصل
- ✅ إعادة ترتيب الأقسام
- ✅ إخفاء/إظهار الأقسام
- ✅ حذف الأقسام

---

## 🚀 البدء السريع

### الخطوة 1: افتح المحرر
```
http://localhost:3000/admin/home-editor
```

### الخطوة 2: استيراد الأقسام الحالية
عند فتح المحرر للمرة الأولى، ستجد خيارين:

1. **استيراد من الصفحة الحالية** (موصى به) 📥
   - يستورد جميع الأقسام الموجودة في الصفحة الرئيسية الحالية
   - يشمل: Hero, Features, Categories, Courses, Stats, Testimonials, CTA
   - جميع المحتويات والبيانات ستكون جاهزة

2. **إضافة قسم مخصص** ➕
   - لإنشاء قسم جديد من الصفر

### الخطوة 3: تحرير الأقسام
بعد الاستيراد، يمكنك:
- ✏️ تعديل العناوين والنصوص
- 🎨 تغيير الألوان
- 🔘 إضافة أو إزالة الأزرار
- 📝 تحرير العناصر الفرعية

### الخطوة 4: حفظ التغييرات
- اضغط على **"حفظ التغييرات"** في الأعلى
- ستظهر رسالة تأكيد عند النجاح

### الخطوة 5: معاينة النتيجة
```
http://localhost:3000/dynamic-home
```

---

## 📦 الأقسام المستوردة

عند الاستيراد، ستحصل على 7 أقسام جاهزة:

### 1. Hero Section 🏠
- **العنوان:** تعلم البرمجة والذكاء الاصطناعي
- **العنوان الفرعي:** مع مستر تامر محفوظ
- **زر:** استكشف الدورات

### 2. Features Section ⭐
6 مميزات جاهزة:
- محاضرات فيديو عالية الجودة 🎥
- ملفات ومراجع شاملة 📄
- شهادات معتمدة 🏆
- مجتمع تفاعلي 👥
- تعلم بالوتيرة المناسبة ⏰
- دعم فني مستمر 🎧

### 3. Categories Section 📚
- **العنوان:** المراحل الدراسية
- **يتم تحميل الفئات من قاعدة البيانات تلقائياً**
- عرض جميع المراحل المنشورة

### 4. Courses Section 📖
- **العنوان:** الدورات التدريبية
- **يتم تحميل الدورات من قاعدة البيانات تلقائياً**
- عرض أول 6 دورات
- زر: عرض جميع الدورات

### 5. Stats Section 📊
4 إحصائيات:
- 500+ طالب نشط 👥
- 15+ دورة تدريبية 📚
- 450+ شهادة صادرة 🏆
- 98% معدل رضا الطلاب 📈

### 6. Testimonials Section 💬
4 آراء طلاب:
- أحمد محمد 👨‍💻
- سارة أحمد 👩‍💻
- محمود علي 🧑‍💻
- فاطمة حسن 👩‍🔬

### 7. CTA Section 📢
- **العنوان:** جاهز لتطوير مهاراتك؟
- **زر:** سجل الآن مجاناً

## ⚙️ إعدادات كل قسم

### إعدادات عامة (كل الأقسام)
- **عنوان القسم** - العنوان الرئيسي
- **العنوان الفرعي** - نص توضيحي
- **المحتوى** - محتوى تفصيلي
- **لون الخلفية** - اختر أي لون
- **لون النص** - اختر أي لون
- **إظهار زر** - تفعيل/إلغاء زر الإجراء
  - نص الزر
  - رابط الزر

### إعدادات العناصر (Features, Stats, Testimonials)
كل عنصر يحتوي على:
- **العنوان/الاسم**
- **الوصف**
- **الأيقونة** (إيموجي)
- **القيمة** (للإحصائيات)

---

## 🎨 التخصيص المتقدم

### تغيير الألوان
```
لون الخلفية: #3B82F6 (أزرق)
لون النص: #FFFFFF (أبيض)
```

### إضافة عنصر جديد في Features
1. ابحث عن قسم Features
2. انتقل إلى "settings.items"
3. أضف عنصر جديد بالحقول:
   - title: "العنوان"
   - description: "الوصف"
   - icon: "🎯"

### إعادة الترتيب
- ⬆️ استخدم السهم لأعلى
- ⬇️ استخدم السهم لأسفل

### إخفاء قسم
- 👁️ اضغط على أيقونة العين
- القسم سيصبح غير نشط (لن يظهر في الصفحة)

---

## 🔄 إعادة الاستيراد

إذا أردت إعادة استيراد الأقسام الافتراضية:

1. احذف جميع الأقسام الحالية
2. زر "استيراد من الصفحة الحالية" سيظهر مرة أخرى
3. اضغط عليه لإعادة الاستيراد

**⚠️ تحذير:** سيتم حذف جميع التعديلات السابقة!

---

## 🛠️ API Endpoints

### استيراد الأقسام الافتراضية
```
POST /api/home-sections/import-default
```

### جلب جميع الأقسام
```
GET /api/home-sections
```

### إضافة قسم جديد
```
POST /api/home-sections
Body: { type, title, subtitle, content, isActive, order, settings }
```

### تحديث قسم
```
PATCH /api/home-sections/[id]
Body: { ...updated fields }
```

### حذف قسم
```
DELETE /api/home-sections/[id]
```

### تحديث جميع الأقسام (للترتيب)
```
PUT /api/home-sections
Body: { sections: [...] }
```

## 💡 نصائح مهمة

1. **احفظ دائماً** بعد كل تعديل
2. **جرّب في المعاينة** قبل النشر
3. **استخدم الألوان المتناسقة** مع هوية الموقع
4. **لا تحذف قسم** إلا إذا كنت متأكداً
5. **الإيموجي أفضل** من الأيقونات للبساطة

## ❓ الأسئلة الشائعة

**س: كيف أضيف صورة للقسم؟**
ج: استخدم حقل `imageUrl` في الإعدادات (قريباً)

**س: هل يمكن نسخ قسم؟**
ج: حالياً لا، لكن يمكن إضافة قسم جديد بنفس المحتوى

**س: ماذا لو أردت ترتيباً مختلفاً؟**
ج: استخدم الأسهم ⬆️⬇️ لإعادة الترتيب

**س: هل التغييرات تطبق فوراً؟**
ج: بعد الحفظ، نعم. الصفحة ستتحدث تلقائياً
