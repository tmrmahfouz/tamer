# 🔍 كيفية التحقق من أخطاء السيرفر

## 📋 الخطوات:

### 1. **افتح نافذة Command Prompt السوداء**
(النافذة التي يعمل فيها `npm run dev`)

### 2. **ابحث عن رسائل الخطأ**
ستجد شيئاً مثل:
```
Create lesson error: [الخطأ]
Error details: [التفاصيل]
```

### 3. **انسخ رسالة الخطأ وأرسلها لي**

---

## 🐛 الأخطاء المحتملة:

### 1. **Token غير صحيح:**
```
JsonWebTokenError: invalid token
```
**الحل:** سجل خروج ثم سجل دخول مرة أخرى

### 2. **Validation Error:**
```
ValidationError: ...
```
**الحل:** تأكد من ملء جميع الحقول المطلوبة

### 3. **Connection Error:**
```
MongooseError: ...
```
**الحل:** تأكد من أن MongoDB يعمل

---

## 🔧 حل سريع:

### جرب إعادة تسجيل الدخول:
1. سجل خروج
2. سجل دخول كـ Admin مرة أخرى:
   ```
   Email: admin@tamermahfouz.com
   Password: admin123
   ```
3. جرب إضافة الدرس مرة أخرى

---

**أخبرني برسالة الخطأ من نافذة CMD!** 🖥️
