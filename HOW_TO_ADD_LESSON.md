# 📚 كيفية إضافة درس جديد للدورة

## 🔐 المتطلبات:

1. ✅ تسجيل دخول كـ Admin أو Instructor
2. ✅ الدورة موجودة
3. ✅ لديك صلاحية على الدورة

---

## 📋 خطوات إضافة درس من لوحة Admin:

### 1. سجل دخول كـ Admin:
```
Email: admin@tamermahfouz.com
Password: admin123
```

### 2. اذهب لإدارة الدورات:
```
http://localhost:3000/dashboard/admin/courses
```

### 3. اختر الدورة التي تريد إضافة درس لها

### 4. اضغط "إضافة درس" أو "Lessons"

### 5. املأ بيانات الدرس:
- **العنوان** (مطلوب)
- **الوصف** (اختياري)
- **المحتوى** (مطلوب)
- **المدة** (مثل: "30 دقيقة")
- **رابط الفيديو** (اختياري)
- **مجاني؟** (نعم/لا)
- **الترتيب** (يتم تلقائياً إذا تركته فارغاً)

### 6. اضغط "حفظ" أو "إضافة"

## 🔧 إذا ظهر خطأ 500:

### تحقق من:

#### 1. **أنت مسجل دخول:**
```javascript
// في Console:
document.cookie
// يجب أن ترى: token=...
```

#### 2. **البيانات المطلوبة:**
- ✅ title (العنوان)
- ✅ content (المحتوى)
- ✅ course (معرف الدورة)

#### 3. **الصلاحيات:**
- ✅ role = 'admin' أو 'instructor'
- ✅ إذا instructor: يجب أن تكون مالك الدورة

## 🐛 تشخيص الأخطاء:

### إذا ظهر خطأ 500:

#### 1. افتح Console في المتصفح (F12)
#### 2. ابحث عن رسالة الخطأ
#### 3. الأخطاء الشائعة:

**خطأ: "غير مصرح"**
- **السبب:** لست مسجل دخول
- **الحل:** سجل دخول كـ Admin

**خطأ: "الدورة غير موجودة"**
- **السبب:** معرف الدورة خاطئ
- **الحل:** تأكد من معرف الدورة

**خطأ: "validation failed"**
- **السبب:** بيانات ناقصة
- **الحل:** تأكد من ملء جميع الحقول المطلوبة

## 🧪 اختبار إضافة درس يدوياً:

### باستخدام Postman أو curl:

```bash
curl -X POST http://localhost:3000/api/courses/69237c10d45260df9ec8f201/lessons \
  -H "Content-Type: application/json" \
  -H "Cookie: token=YOUR_TOKEN_HERE" \
  -d '{
    "title": "درس جديد",
    "description": "وصف الدرس",
    "content": "محتوى الدرس",
    "duration": "20 دقيقة",
    "isFree": false
  }'
```

## 💡 نصائح:

### 1. **الترتيب التلقائي:**
إذا لم تحدد `order`، سيتم إضافة الدرس في النهاية تلقائياً

### 2. **الدروس المجانية:**
ضع `isFree: true` للدروس التي تريدها مجانية

### 3. **المحتوى:**
يمكنك استخدام Markdown في المحتوى:
```markdown
# عنوان
## عنوان فرعي
- نقطة 1
- نقطة 2
```

### 4. **الفيديو:**
إذا أضفت رابط فيديو، سيظهر في صفحة الدرس
