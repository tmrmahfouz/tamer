# 📚 دليل إدارة الدروس أثناء تعديل الدورة

## 🎯 نظرة عامة

تم إضافة خاصية إدارة الدروس بشكل كامل داخل صفحة تعديل الدورة، مما يتيح للمدرب أو الأدمن إضافة وتعديل وحذف الدروس بسهولة دون الحاجة للانتقال لصفحات أخرى.

---

## 🚀 الميزات الجديدة

### ✅ 1. عرض جميع الدروس
- قائمة منظمة لجميع دروس الدورة
- عرض ترتيب كل درس
- عرض معلومات الدرس (عنوان، مدة، نوع المحتوى)
- إشارة للدروس المجانية

### ✅ 2. إضافة درس جديد
- نموذج شامل لإضافة الدرس
- حقول متعددة (عنوان، وصف، محتوى، فيديو، مدة، ترتيب)
- خيار جعل الدرس مجاني

### ✅ 3. تعديل الدروس
- تعديل أي درس موجود
- تحديث جميع معلومات الدرس
- حفظ التغييرات فوراً

### ✅ 4. حذف الدروس
- حذف أي درس مع تأكيد
- تحديث تلقائي للقائمة

## 📝 خطوات إضافة درس

### 1. افتح صفحة تعديل الدورة:
```
/dashboard/courses/[course_id]/edit
```

### 2. اضغط على "إضافة درس":
- الزر موجود في أعلى قسم الدروس

### 3. املأ بيانات الدرس:

#### الحقول الإلزامية:
- **عنوان الدرس** ⭐ (مطلوب)

#### الحقول الاختيارية:
- **وصف الدرس**: وصف مختصر للدرس
- **محتوى الدرس**: المحتوى النصي الكامل
- **رابط الفيديو**: رابط YouTube أو أي منصة فيديو
- **المدة**: مثل "15 دقيقة" أو "30 دقيقة"
- **الترتيب**: رقم ترتيب الدرس (يتم تعيينه تلقائياً)
- **درس مجاني**: ✓ لجعل الدرس متاح للجميع

### 4. احفظ الدرس:
- اضغط "إضافة الدرس"
- سيتم إضافة الدرس للقائمة فوراً

---

## ✏️ خطوات تعديل درس

### 1. في قائمة الدروس:
- اضغط على أيقونة القلم (✏️) بجانب الدرس

### 2. عدّل البيانات:
- جميع الحقول قابلة للتعديل
- يمكنك تغيير الترتيب
- يمكنك جعل الدرس مجاني أو مدفوع

### 3. احفظ التعديلات:
- اضغط "تحديث الدرس"
- سيتم تحديث الدرس فوراً

---

## 🗑️ خطوات حذف درس

### 1. في قائمة الدروس:
- اضغط على أيقونة سلة المهملات (🗑️) بجانب الدرس

### 2. تأكيد الحذف:
- سيظهر تأكيد: "هل أنت متأكد من حذف هذا الدرس؟"
- اضغط "موافق" للحذف

### 3. النتيجة:
- سيتم حذف الدرس نهائياً
- سيتم تحديث القائمة تلقائياً

## 🎯 حالات الاستخدام

### 1. إنشاء دورة جديدة:
```
1. أنشئ الدورة من /dashboard/courses/new
2. احفظ الدورة
3. اذهب لصفحة التعديل
4. ابدأ بإضافة الدروس واحداً تلو الآخر
```

### 2. تحديث دورة موجودة:
```
1. افتح صفحة تعديل الدورة
2. شاهد الدروس الموجودة
3. أضف دروس جديدة أو عدّل الموجودة
4. احذف الدروس غير المطلوبة
```

### 3. إعادة ترتيب الدروس:
```
1. افتح تعديل كل درس
2. غيّر رقم الترتيب
3. احفظ التعديلات
4. الدروس ستظهر بالترتيب الجديد
```

## 🔄 سير العمل الموصى به

### لإنشاء دورة كاملة:

#### المرحلة 1: إنشاء الدورة
```
1. اذهب إلى /dashboard/courses/new
2. املأ معلومات الدورة الأساسية
3. احفظ الدورة
```

#### المرحلة 2: إضافة الدروس
```
1. اذهب لصفحة تعديل الدورة
2. أضف الدرس الأول (مقدمة - مجاني)
3. أضف باقي الدروس بالترتيب
4. تأكد من ترقيم الدروس بشكل صحيح
```

#### المرحلة 3: المراجعة
```
1. راجع جميع الدروس
2. تأكد من المحتوى والروابط
3. حدد الدروس المجانية
4. احفظ التغييرات
```

#### المرحلة 4: النشر
```
1. في نموذج تعديل الدورة
2. فعّل خيار "منشورة"
3. احفظ التغييرات
4. الدورة أصبحت متاحة للطلاب
```

## 🛠️ الـ APIs المستخدمة

### 1. جلب دروس الدورة:
```typescript
GET /api/courses/[id]/lessons

Response:
{
  success: true,
  count: 5,
  lessons: [...]
}
```

### 2. إضافة درس جديد:
```typescript
POST /api/courses/[id]/lessons

Body:
{
  title: "عنوان الدرس",
  description: "وصف الدرس",
  content: "محتوى الدرس",
  videoUrl: "https://...",
  duration: "15 دقيقة",
  order: 1,
  isFree: false
}

Response:
{
  success: true,
  message: "تم إضافة الدرس بنجاح",
  lesson: {...}
}
```

### 3. تعديل درس:
```typescript
PUT /api/lessons/[id]

Body:
{
  title: "عنوان محدث",
  ...
}

Response:
{
  success: true,
  message: "تم تحديث الدرس بنجاح",
  lesson: {...}
}
```

### 4. حذف درس:
```typescript
DELETE /api/lessons/[id]

Response:
{
  success: true,
  message: "تم حذف الدرس بنجاح"
}
```
