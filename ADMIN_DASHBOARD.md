## 📊 APIs الإدارية

### Overview API
```
GET /api/admin/overview

Response:
{
  success: true,
  stats: {
    revenue: { total, growth },
    students: { total, growth },
    courses: { total, growth },
    enrollments: { total, growth },
    today: { newStudents, newEnrollments },
    pending: { payments, tickets },
    revenueChart: [...],
    topCourses: [...],
    online: { users },
    live: { sessions }
  }
}
```

### Students API
```
GET    /api/admin/students
POST   /api/admin/students
PUT    /api/admin/students/[id]
DELETE /api/admin/students/[id]
```

### Enrollments API
```
GET  /api/admin/enrollments
POST /api/admin/enrollments/manual
POST /api/admin/enrollments/[id]/approve
POST /api/admin/enrollments/[id]/reject
```

### Payments API
```
GET  /api/admin/payments
POST /api/admin/payments/[id]/verify
POST /api/admin/payments/[id]/reject
```

### Analytics API
```
GET /api/admin/analytics?period=month
GET /api/admin/statistics?period=month
GET /api/admin/reports?type=overview&format=json
```
